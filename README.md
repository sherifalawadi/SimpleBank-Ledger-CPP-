Simple Bank Ledger (C++)A robust command-line interface (CLI) application designed to manage bank accounts and financial records with high precision. This project focuses on Data Persistence using CSV files and ensures financial integrity through a No-Overdraft policy.🚀 Key FeaturesAccount Management: Create unique accounts with an optional initial deposit.Transaction Suite: Supports standard banking operations including Deposit, Withdraw, and Transfer between accounts.Persistent Storage: Data is automatically saved to and loaded from data/accounts.csv and data/ledger.csv.Detailed Logging: Every transaction is recorded with a unique 10-digit ID and an ISO-8601 formatted timestamp.Financial Safety: Strict validation to prevent account overdrafts.Reporting: Generate real-time account statements and balance inquiries.📂 Data Structure (CSV)The system maintains data in a structured format for easy auditing:Accounts File (accounts.csv)ColumnDescriptionaccount_idUnique identifier for the user.nameAccount holder's name.balance_centsCurrent balance stored in cents to ensure precision.Ledger File (ledger.csv)ColumnDescriptionts_isoISO-8601 Timestamp.tx_id10-digit unique transaction ID.typeTransaction type (DEPOSIT, WITHDRAW, TRANSFER).from / toOrigin and destination account IDs.amount_centsTransaction value in cents.🛠️ CLI CommandsThe application supports the following command syntax:help: Show available commands.list: Display all registered accounts.create <id> <name> <initial_amount>: Open a new account.deposit <id> <amount>: Add funds to an account.withdraw <id> <amount>: Remove funds (if balance permits).transfer <from_id> <to_id> <amount>: Move funds between two accounts.balance <id>: Check current account balance.statement <id>: Print transaction history for an account.quit: Safely exit the application.
